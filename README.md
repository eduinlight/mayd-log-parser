## Build
```BASH
npm run build
```

## Run
```BASH
#run with script generated
./dist/log-parser -i input.log -o output.json
./dist/log-parser --input input.log --output output.json
#run with node
node ./dist/index.js -i input.log -o output.json
#or
node . -i input.log -o output.json
```

## Run test
```BASH
npm run test
```
